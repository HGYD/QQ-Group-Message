# QQ-Group-Message
本程序属于 个人定制 
主要针对 获取个人账户的QQ群消息
同时 可以获取群列表 和 群成员 及其详细信息（相对的）

程序在编写前，主要参考的代码是：

https://github.com/zeruniverse/QQRobot

当然，它的功能很强大，而我要实现的不需要那么多

在具体实现上：
复用了QQRobot：
    HttpClient 类 
    qq登录的函数（有修改）
    对于心跳包的处理（有修改）

个人添加的部分：
    对群消息的特定接收
    对群消息的分类存储
    获取成员列表
    获取群列表
    获取成员的详细信息（这个接口 通过qq客户端抓包获得 ）
    
    
另外，webqq上的一些js加密函数会时常更新，导致需要不断修改。
